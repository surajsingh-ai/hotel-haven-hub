# Welcome to your Lovable project
Hotel Haven Hub
Hotel Haven Hub is a responsive hotel booking landing page built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui. It presents a premium hotel discovery experience with featured listings, testimonials, travel inspiration, and a polished modern UI suitable for hospitality or travel-focused front-end portfolios.

Features
Modern responsive landing page for hotel discovery and booking inspiration

Hero section with strong visual branding and call-to-action buttons

Featured hotel cards with ratings, pricing, amenities, and booking prompts

Destination and travel inspiration sections to improve user engagement

Testimonial and trust-building sections for better product storytelling

FAQ section for common booking-related questions

Reusable UI components powered by shadcn/ui and Radix primitives

Fast development workflow with Vite and TypeScript

Tech Stack

Tailwind CSS

Getting Started
Prerequisites
Make sure you have one of the following installed:

Node.js 18+

npm

Installation
bash
git clone https://github.com/surajsingh-ai/hotel-haven-hub.git
cd hotel-haven-hub
npm install
Run Locally
bash
npm run dev
The Vite dev server runs on:

bash
http://localhost:8080
Build for Production
bash
npm run build
Preview Production Build
bash
npm run preview
Lint the Codebase
bash
npm run lint
Run Tests
bash
npm run test
Available Scripts
Script	Description
npm run dev	Starts the Vite development server
npm run build	Creates the production build
npm run build:dev	Builds the app in development mode
npm run preview	Previews the production build locally
npm run lint	Runs ESLint
npm run test	Runs tests once with Vitest
npm run test:watch	Runs Vitest in watch mode
